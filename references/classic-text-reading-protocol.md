# EMARX 经典文本精读协议

## 一、何时需要精读经典文本

以下情况必须在写作前安排经典文本精读：

- 论文核心概念直接来自马克思主义经典（如异化、实践、人民主体性、剩余价值、意识形态等）；
- 论文要回应的争论涉及对经典文本的不同解读；
- 论文要批评某种被经典作家直接回应过的思潮或观点；
- 论文的"理论视角"声称以某部经典为依据；
- 题目中出现"基于《×××》""以×××为中心"等文本个案表述。

经典文本包括但不限于：马克思、恩格斯、列宁、毛泽东等人的著作、讲话、书信，以及被论文明确作为理论依据的其他思想家核心文本。

## 二、精读什么

精读不是摘抄名言，而是回答以下问题：

1. **核心论断**：该文本针对什么问题提出了什么判断？判断的精确表述是什么？
2. **理论语境**：这个判断是在回应谁、批评什么、解决什么理论困难？
3. **概念边界**：文本中关键概念（如"实践""主体性""人民"）是在什么意义上使用的？与日常用法或后世用法有何不同？
4. **论证运动**：作者如何从前提推到这个判断？中间经过了哪些环节？
5. **历史条件**：这个判断是在什么社会历史条件下形成的？哪些条件限制了它的适用范围？
6. **与本文的连接**：这个判断对本文题目有什么解释力？它能说明本文对象的哪个方面？它的限度在哪里？

## 三、怎么进入正文

经典文本进入正文有三种方式，按学理性从低到高排列：

### 1. 语录式引用（最低，尽量避免）

> 马克思指出，"全部社会生活在本质上是实践的"[1]。

问题：只摆权威，不解释这句话对本文论证的具体作用。容易变成装点门面。

### 2. 概念来源式进入

> 马克思把实践理解为感性的人的活动，以此区别于直观的唯物主义[1]。沿此视角看，智能传播中的内容生成不能被仅理解为技术操作，而应被理解为嵌入社会关系中的感性活动。

优点：说明经典概念如何界定本文对象。仍较浅。

### 3. 论证参与式进入（推荐）

> 马克思批评旧唯物主义"只是从客体的或者直观的形式去理解"对象，而要求把对象理解为"人的感性活动"[1]。这一区分对理解智能生成技术有重要意义：若把大模型仅看作信息处理工具，便停留在直观形式；只有把它放回人机互动的感性实践，才能看清意义如何在生成中被重新组织。

优点：经典论断被用来揭示本文对象的特定关系，经典文本真正参与论证推进。

## 四、操作步骤

1. **确定必读经典**：根据题目，从马克思主义基本原理或相关思想史中确定 1-3 部核心文本。
2. **回到原文**：优先读原始文本的完整段落，不是只读二手概述或名言汇编。
3. **写精读笔记**：按"核心论断—理论语境—概念边界—论证运动—历史条件—与本文连接"六条记录。
4. **建立功能绑定**：明确该经典文本在论文中的具体功能：
   - 提供概念框架
   - 提供批判靶子
   - 提供理论限度
   - 提供与当代问题的连接点
5. **写入正文**：选择"论证参与式"进入方式，让经典论断与本文对象形成具体关系。

## 五、常见错误

- **断章取义**：只引对本文有利的半句话，忽略原文的限定条件。
- **时空错位**：把经典作家在特定历史条件下的判断直接套用到当代问题，不做转化。
- **权威标签化**：把经典引用当装饰，引用前后论证逻辑不变。
- **过度解读**：把经典文本中没有的意思强加进去。
- **只引不释**：引用经典原话后，不说明它如何支撑本文判断。

## 六、检查清单

经典文本进入正文前，确认：

- [ ] 引用是否来自原文完整段落，不是二手转述；
- [ ] 是否说明了该论断的理论语境；
- [ ] 是否说明了关键概念在本文中的用法；
- [ ] 是否说明了该论断与本文对象的具体关系；
- [ ] 删除该引用后，本段论证是否会明显减弱；
- [ ] 是否存在断章取义或时空错位。
