# EMARX 写作方法选择器

## 一、用途

本文件把题目类型、段落任务与 `references/ten-writing-methods.md` 中的十种写作方法对应起来，帮助在选题和生成阶段快速确定应使用哪种或哪几种方法组合。它不是强制模板，而是减少主模型临场判断成本的导航工具。

## 二、题目类型 → 方法组合映射

以下映射基于马理论论文常见题型，结合方法论选择指南和十种写作方法设计。实际写作时应根据具体题目灵活调整。

| 题目类型 | 典型特征 | 主导方法 | 辅助方法 | 慎用方法 |
|---|---|---|---|---|
| 经典文本阐释 | 以一部经典著作为核心对象 | 方法 3（经典互文）+ 方法 9（概念界定） | 方法 7（历史条件分析）+ 方法 6（论-据-证闭环） | 方法 8（物质利益揭示）、方法 10（终于实践） |

## 三、段落任务 → 方法映射

同一篇论文的不同段落可能使用不同方法。根据段落任务选择：

| 段落任务 | 推荐方法 | 不推荐方法 |
|---|---|---|
| 开篇点题 | 方法 1（开宗明义） | 方法 7、8、9（过于具体） |
| 文献综述/理论对话 | 方法 5（批判-建构）+ 方法 6（论-据-证闭环） | 方法 10（不要提前下实践结论） |
| 概念界定 | 方法 9（概念界定） | 方法 5、8（避免批判先行） |
| 机制分析 | 方法 6（论-据-证闭环）+ 方法 4（对立统一） | 方法 1、10（避免口号和实践跳跃） |
| 历史追溯 | 方法 2（逻辑与历史统一）+ 方法 7（历史条件分析） | 方法 10（避免以今度古） |
| 案例/材料分析 | 方法 6（论-据-证闭环） | 方法 1、10（避免脱离材料空泛发挥） |
| 批判回应 | 方法 5（批判-建构）+ 方法 6（论-据-证闭环） | 方法 9（单独使用会显得概念空转） |
| 矛盾揭示 | 方法 4（对立统一）+ 方法 6（论-据-证闭环） | 方法 10（避免急于给出实践答案） |
| 路径/对策推导 | 方法 10（始于问题，终于实践）+ 方法 6（论-据-证闭环） | 方法 1、9（避免重新定义问题） |
| 结论收束 | 方法 10（始于问题，终于实践）+ 方法 6（论-据-证闭环） | 方法 5、8（避免引入新批判） |

## 四、方法组合使用原则

1. **一个段落通常以 1-2 种方法为主导**。方法不是标签，不要为了展示方法而堆砌方法。
2. **主导方法服务于段落任务**。选择方法前必须先明确本段要完成的论证动作。
3. **方法之间要有主次**。例如"批判-建构"段落中，批判是手段，建构是目的；不能批判完后没有正面判断。
4. **根据目标期刊调整方法组合**。《马克思主义研究》偏好方法 3、4、5；《求是》偏好方法 1、7、10；《中共党史研究》偏好方法 2、7、6。
5. **避免方法滥用**。方法 8（物质利益揭示）不是处处适用；方法 10（终于实践）不能变成空洞对策。

## 五、方法选择快速流程

为每个小节或段落选择方法时，按以下顺序判断：

```text
1. 本小节/段落的任务是什么？（定义、解释、批判、机制分析、案例、路径、结论）
2. 本题属于哪种题目类型？（查表二）
3. 该任务下推荐哪种方法？（查表三）
4. 选择 1-2 种主导方法，必要时搭配 1 种辅助方法。
5. 说明选择理由：为什么这种方法最匹配当前任务？
6. 生成段落时按选定方法组织语言运动。
```

## 七、常见错误

1. **所有段落都用同一种方法**：如全文都是"不是……而是……"的批判-建构，导致单调。
2. **为使用方法而制造虚假对立**：没有真实理论对手，硬套方法 5。
3. **方法与实践脱节**：选择了方法 10（终于实践），但结论只是重复政策口号。
4. **方法堆砌**：一个段落同时套用 3-4 种方法，论证主线模糊。
5. **方法选择不与题目类型匹配**：如经典文本阐释类文章过度使用方法 8（物质利益揭示）。
