# EMARX 政治合规协议

## 一、适用范围

本协议适用于所有拟投向马克思主义理论类期刊、党报理论版、高校学报马克思主义栏目及相近政治性学术场域的论文。政治合规是马理论顶刊审稿的第一关，不能通过后期润色弥补，必须在选题、起草、审稿全过程中主动把关。

## 二、政治方向自检清单

动笔前和定稿后必须逐项确认：

- [ ] 论文是否坚持马克思主义指导地位，是否体现习近平新时代中国特色社会主义思想的世界观和方法论；
- [ ] 论文是否与中国共产党的基本理论、基本路线、基本方略保持一致；
- [ ] 论文是否准确使用党的创新理论核心表述，未出现自行简化、曲解或时尚化改写；
- [ ] 论文对重大历史事件、历史人物、党史军史国史的评价是否符合党中央权威结论；
- [ ] 论文是否避免以"中立化""去政治化"姿态削弱意识形态属性；
- [ ] 论文引用的领导人讲话、中央文件是否来自权威原文，而非二手概述或网络摘录；
- [ ] 论文是否没有出现可能被视为影射、调侃、消解或质疑党的领导与中国特色社会主义制度的表述；
- [ ] 涉及民族、宗教、港澳台、国际关系、党史军史等敏感议题时，是否与现行法律法规和国家立场一致。

## 三、核心政治表述规范

### 1. 党的创新理论表述

以下表述必须规范、完整，不得自行替换或缩写：

- "习近平新时代中国特色社会主义思想"（完整表述，不得简化为"习近平思想""习思想"等）；
- "中国式现代化"（注意与"现代化""西方现代化"的区分边界）；
- "全过程人民民主"（不使用"全过程民主"等省略形式）；
- "两个确立""两个维护"（作为固定政治概念使用，不得泛化或戏仿）；
- "五位一体"总体布局、"四个全面"战略布局；
- "新质生产力"（准确理解其政治经济学内涵，避免与泛化的"新技术""新产业"混用）。

### 2. 历史与人物表述

- 党史重大事件评价以《中共中央关于党的百年奋斗重大成就和历史经验的决议》等权威文件为准；
- 对毛泽东、邓小平等历史人物的评价，使用党中央正式表述，不得出现片面化、碎片化、情绪化判断；
- 涉及改革开放前后两个历史时期，使用"相互联系又有重大区别""不能相互否定"等权威论断。

### 3. 价值立场表述

- 避免以"价值中立""多元视角"模糊马克思主义立场；
- 避免把"人民""群众"等概念空洞化、口号化；
- 涉及西方理论、思潮、概念时，必须进行批判性吸收，不得直接移植其意识形态预设。

## 四、政策引用核查流程

论文中每出现一次政策、文件、讲话、会议精神引用，必须按以下流程核验：

1. **定位原文**：优先使用《求是》、新华社、人民日报、中国政府网、中共中央办公厅/国务院办公厅公开发布的原文；
2. **核对版本**：确认引用的是最新版本，旧文件已被修订时应注明；
3. **核对表述**：引号内文字必须与原文完全一致，不得拼接、改写、断章取义；
4. **核对语境**：不能把针对特定场合的讲话套用到不相关议题上；
5. **记录出处**：在 `source-claim-map` 中标注原文 URL、文件名称、发布日期、具体段落。

## 五、敏感表述与禁用词

### 禁用表述

以下表述在论文中原则上不得出现：

- "党天下""一党专政""专制""独裁"等污名化表述；
- 对党和国家领导人使用不尊称、戏称、洋称；
- "普世价值""宪政民主""三权分立""多党制"等未经批判性限定直接使用；
- 把中国特色社会主义制度与"资本主义""威权主义"简单类比；
- 对重大历史事件使用境外敌对势力或网络流行叙事框架。

### 需谨慎表述

以下表述若使用，必须给出明确界定和学术论证：

- "意识形态斗争""颜色革命""和平演变"等，需说明具体语境；
- "左""右""保守""激进"等政治标签，需避免简单套用；
- "西方马克思主义""后现代""新自由主义"等概念，需批判性说明其理论来源和适用范围。

## 六、常见政治性错误

1. **理论空转**：大段引用党的创新理论但不进入具体问题，把论文写成政策宣讲；
2. **政治替代学理**：只讲政治正确，没有学理论证，使论文失去学术性；
3. **偷偷弱化**：以"客观研究""中性分析"为名，把马克思主义立场边缘化；
4. **时空错位**：用今天的政策语言直接评判历史事件，或用历史语境消解当代政治判断；
5. **引文失真**：把领导人讲话、中央文件掐头去尾，服务于本文预设观点；
6. **概念混用**：把"中国式现代化"与西方现代化混为一谈，把"全过程人民民主"简单等同于参与式民主。

## 七、政治合规审核步骤

### 写作前

- 明确论文的政治定位：本文要在哪个理论问题上推进党的创新理论的学理化阐释？
- 识别敏感议题：是否涉及党史、民族、宗教、港澳台、国际关系、意识形态斗争？

### 起草中

- 每段写完后自问：这段话如果发表在顶刊上，是否会被质疑政治方向？
- 涉及政策、讲话、文件时，立即标注需要核验的出处。

### 定稿前

- 运行 `scripts/political_audit.py`（待开发）做机械扫描：禁用词、敏感表述、领导人姓名规范、核心政治概念完整性；
- 主模型逐段复核政治立场和政策引用准确性；
- 对高风险表述，必须回到权威原文重新核对。

## 八、与学理性的关系

政治合规不是降低学理性，而是为学理性划定安全边界。马理论顶刊要求的是：

- **政治方向正确** + **学理阐释深入** + **现实问题导向**

三者缺一不可。不能为了政治安全而写成口号文章，也不能为了显示学术性而模糊政治立场。

## 九、失败信号

出现以下情况必须重写：

- 论文可以被理解为对党的基本理论、基本路线提出质疑；
- 政策引文与原文明显不符；
- 为追求"新颖"而使用可能引发政治争议的网络流行概念；
- 把西方理论直接作为评价中国实践的默认标准；
- 结论部分只剩下政治口号，没有任何学术判断。

## 十、交付标准

完整论文交付时，政治合规状态必须明确报告：

- 是否涉及敏感议题；
- 核心政策引文是否已回到权威原文核验；
- 政治审计脚本是否通过；
- 是否存在政治表述上的残余风险。
