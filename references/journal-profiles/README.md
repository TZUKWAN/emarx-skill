# EMARX 期刊精准画像

本目录收录马克思主义理论及相关领域主要期刊的精准画像，用于在选题、起草和投稿阶段匹配目标期刊风格。

## 已收录期刊

| 期刊 | 定位 | 主要风格 |
|---|---|---|
| 马克思主义研究 | 马理论综合性权威月刊 | 思想性、理论性、批判性、创新性并重 |
| 马克思主义与现实 | 中央党史和文献研究院主办 | 马克思主义理论与当代现实结合，重视国外马克思主义 |

## 使用方式

1. **选题阶段**：根据用户目标期刊或论文主题，读取对应画像，明确该刊的选题偏好和篇幅体例；
2. **起草阶段**：按照画像中的摘要风格、引言模式、标题特征和引注规范组织论文；
3. **审稿阶段**：对照画像中的"写作禁忌"和"范文特征"检查稿件匹配度；
4. **投稿阶段**：参考画像中的审稿与投稿信息，合理安排投稿计划。

## 注意事项

- 画像基于公开信息和编辑讲座整理，具体投稿要求以期刊官方最新通知为准；
- 画像只用于后台控制，不进入论文正文；
- 用户未指定目标期刊时，优先以《马克思主义研究》或最接近论文主题的期刊画像为默认参照。
