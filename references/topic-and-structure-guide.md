# EMARX 选题与结构轻量指南

本文件吸收 marx-paper-skills 的选题与结构方法，为 EMARX 的审题、拟题和结构环节提供更轻、更直接的操作参考。所有规律都来自已发表论文的归纳，具体论文应根据对象内部关系灵活调整，不得机械套用。

## 一、选题：三层法

### 第一步：认识优质题目的特征

#### 1. 标题骨架（5 种）

- **主标题 + 破折号副标题**：主标题立问题/论点，副标题限范围/材料/视角。
  - 例：论唯物史观传播的本土实践与革命效应——以蔡和森《社会进化史》为中心
- **冒号式**：冒号前凝练核心概念，冒号后展开限定或视角。
  - 例：动之以情：《晋察冀日报》在大生产运动中的情感动员研究（1938-1945）
- **从 X 到 Y 演进式**：揭示转变、演进或发展过程。
  - 例：从交往异化到劳动异化：对赫斯与青年马克思文本中 erwerben 的考察
- **陈述直陈式**：平实陈述研究对象与内容。
  - 例：论马克思对德国观念论 Recht 范畴的批判与重构
- **问句/反问式**：用问号抛出核心问题。EMARX 默认避免标题用问号，除非用户明确要求或目标期刊常用。

#### 2. 研究路数（6 类）

- **概念史/话语史**：源流、演变、定型、谱系、生成逻辑。
- **文本/思想个案**：以一部经典文本或一位思想家的某一思想为锚。
- **历史事件/运动/制度**：以一段具体历史为对象，总结经验或逻辑。
- **理论批判/辨析**：回应理论争议，辨析概念学理内涵。
- **现实应用/路径创新**：面向当代现实问题，探讨怎么办。
- **比较/演进**：揭示转变、对话或张力。

#### 3. 深层气质

- 大背景 + 小切口；
- 问题意识鲜明；
- 材料/范围明确限定；
- 有理论框架/视角；
- 学术张力（X 与 Y、从 X 到 Y、何以可能）；
- 主副分工清晰。

### 第二步：情况确认与分流

拟题前先确认三件事：

1. **场景**：课程论文 / 期刊投稿 / 学位论文。
2. **资料**：有本地文献 / 无文献。
3. **方向明确度**：很明确 / 大概方向 / 没方向。

- **有资料路径**：读资料 → 提炼方向 → 与用户确认 → 生成候选题目。
- **没资料路径**：通过五个提问锚点（对象、矛盾、视角、材料、边界）把模糊想法具体化。

### 第三步：拟题配方

按“方向 × 路数 × 骨架 × 切口”生成一批候选题目，筛选后留 10 个左右让用户选。不要堆砌范式标记词，一个好题目通常只用一两处学术标识。

## 二、结构搭建

### 默认 scaffold（可缩放）

参考《马克思主义研究》等顶刊样本：

- 引言：350-600 字；
- 主体：3-5 个一级标题，常见 3 大标题（约 54%）、4 大标题（约 31%）；
- 每个大标题下先有 100-250 字引子段，再拆 2-3 个二级标题；
- 每个二级小标题段约 800-1400 字，中心约 1100 字；
- 结语：约 500 字，可选——顶刊样本中约 73% 无独立结语；
- 完整论文：10000-12000 字。

### 大标题起法

1. **递进式排比**：最主流。三个大标题用同一句式对仗排开，层层递进。
   - 例：基于主体视域 → 基于历史视域 → 基于前瞻视域
2. **概念-展开式（冒号式）**：冒号前核心概念，冒号后展开方向。
3. **从 X 到 Y 式**：揭示演进或转换。
4. **直接陈述式**：平实陈述该部分内容。

### 小标题起法

- **冒号式拆解**：维度/层面 + 具体内容。
- **从 X 到 Y 式**：揭示转换、递进或对立。

小标题之间最好有内在递进关系，不是随机罗列。

### 结构防火墙

不得把后台研究动作做成正式标题：

- 研究对象与概念边界
- 概念界定
- 理论框架
- 创新点分析
- 路径建设与可执行条件

概念边界、理论框架等应嵌入引言或具体论证段落。

### 标题禁用

- 标题不得使用问号；
- 标题不得使用“如何、何以、为何、为什么、怎样、怎么、是否、能否、吗”等题目化提问腔。

## 三、灵活原则

- 结构由题目内部的真实关系生长出来，不是把“机遇、挑战、路径”机械拆成三章。
- 前一部分应产生后一部分的论证必要性；删去任一部分后全文论证受影响。
- 若主体两部分互换顺序仍然成立，说明结构只是并列罗列，需要重新设计。
