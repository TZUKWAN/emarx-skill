# EMARX Nuwa 蒸馏：工作空间论文写作模式

本目录使用 `huashu-nuwa` 的蒸馏方法对工作空间 `D:/BAOXUE/.codex/pdf_text` 中的中文学术论文进行多维度分析，目的是让 EMARX 的生成逻辑从“套模板”转向“模仿成熟论文的论证方式”。

## 蒸馏维度

| 文件 | 维度 | 核心关注点 |
|---|---|---|
| `research/01-argument-structure.md` | 论证结构 | 问题提出、结构范式、理论嫁接、案例因果 |
| `research/02-source-citation.md` | 文献引用 | 政策问题化、理论功能化、案例可核实、综述缺口化 |
| `research/03-paragraph-moves.md` | 段落推进 | 理论降维、概念分层、案例回扣、边界句式 |
| `research/04-concept-theory.md` | 概念理论 | 操作定义、机制化、成对张力、理论本土化 |
| `research/05-title-abstract-headings.md` | 标题摘要层级 | 标题三要素、摘要核心命题、结构张力驱动 |
| `research/06-failure-patterns.md` | 失败模式 | 政策复述、概念罗列、案例贴标签、对策空泛 |

## 使用方式

EMARX 在生成和审稿时应把这些发现当作语料规律，而不是新增禁令。具体规则已经写回：

- `references/argument-depth-protocol.md`
- `references/citation-protocol.md`
- `references/material-anchoring-protocol.md`
- `references/paragraph-moves-protocol.md`
- `scripts/bad_draft_audit.py`

当用户再次要求“用女娲蒸馏”或“根据工作空间论文优化技能”时，优先复用本目录的研究结果，而不是重新全量扫描。
